# mcp-docs

Documentation for the OKFN MCP platform, published at
https://okfn.github.io/mcp-docs/

Built with [MkDocs](https://www.mkdocs.org/) and the
[Material](https://squidfunk.github.io/mkdocs-material/) theme.

## Local preview

```bash
uv sync
uv run mkdocs serve    # http://127.0.0.1:8000
```

Pushing to `main` builds and deploys the site to GitHub Pages via
GitHub Actions.

See the [contributing page](docs/contributing.md) for house rules
(small files, human-first writing).
