# Open data resources

Beyond tools, the pilot exposed **MCP resources** following the standard,
and added a tool that lets the user find out which resources are
available and consume them.

This was a positive: it invites users to keep analysing the data outside
our tool. The chat is a starting point, not a walled garden. Someone can
follow a link from an answer to the underlying resource and continue on
their own, with a spreadsheet, a notebook, or whatever they prefer.

It also reinforces the traceability idea: the data is not trapped inside
the assistant, it points back out to the open portal it came from.
